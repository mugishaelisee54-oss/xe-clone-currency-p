
# 💱 xe-clone-currency-p

A simple currency converter app that works like XE.com. Convert money between different currencies easily.

**Created by:** Mugisha Elise

**Live Demo:** https://mugishaelisee54-oss.github.io/xe-clone-currency-p/

---

## 📱 What This App Does

- Convert money from one currency to another
- See live exchange rates
- Check your past conversions
- View currency details
- Change app settings

---

## 📁 Files in This Project

- `index.html` - Home page
- `converter.html` - Convert currencies
- `details.html` - View currency info
- `history.html` - See past conversions
- `settings.html` - Change app settings
- `script.js` - Makes everything work

---

## 🛠️ How It Works

1. Pick a currency (like USD)
2. Pick another currency (like EUR)
3. Enter an amount
4. Get the converted amount instantly

---

## 🚀 How to Use

**Option 1:** Click the live demo link above

**Option 2:** Run locally
- Download or clone this project
- Open `index.html` in your browser

---

## ✅ Features

- Works on phone, tablet, and computer
- Simple and clean design
- Real exchange rates
- Saves your conversion history
